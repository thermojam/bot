БОТ В ДАННЫЙ РАБОТАТЕТ ПОСРЕДСТВОМ

ССЫЛКА

Запуск бота с PM2 Чтобы запустить вашего бота, используйте следующую команду: 'pm2 start bot.js --name bot' Здесь --name bot задает имя вашему процессу, что упрощает его идентификацию.

Просмотр сессий Чтобы просмотреть все запущенные процессы, используйте: 'pm2 list' Эта команда покажет вам список всех процессов, запущенных с помощью pm2, их статус, PID и другую информацию.

Остановка процесса Чтобы остановить процесс, используйте: 'pm2 stop bot' Замените bot на имя вашего процесса или используйте PID, если вам это удобнее.

Запуск процесса Чтобы запустить остановленный процесс, используйте: 'pm2 start bot'

Перезапуск процесса Если вы внесли изменения в код вашего бота и хотите перезапустить его, используйте: 'pm2 restart bot'

Удаление процесса Если вам нужно удалить процесс из pm2, используйте: 'pm2 delete bot'

Просмотр логов Чтобы просмотреть логи вашего бота, используйте: 'pm2 logs bot' Это покажет вам вывод вашего бота в реальном времени. Вы можете использовать Ctrl + C, чтобы выйти из режима просмотра логов.

Получение информации о процессе: 'pm2 info bot'
